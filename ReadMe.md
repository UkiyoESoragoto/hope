### 使用说明 / How to use


按下 Cmd + r 加速（刷新）

按下 Cmd + , 打开设置

按下 Cmd + q 退出

请在Linux系统下使用Ctrl键替换Cmd

Windows下没有快捷键

Press Cmd + r to speed up(refresh)

Press Cmd + , to open settings

Press Cmd + q to quit

Use Ctrl key to replace Cmd on Linux
